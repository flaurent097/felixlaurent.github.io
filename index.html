Project Description

Ladder is a memory diary and mini-game app that helps caregivers support loved ones with dementia in maintaining cognitive function.

Through the Memory Album feature, caregivers capture and categorize daily moments and special memories in photo albums. Each photo can include captions or voice notes, helping loved ones recall stories with personal and emotional context, making memories more vivid.

Ladder also includes a Puzzle Memory Game, turning album photos into puzzles that engage cognitive skills. Upon puzzle completion, captions are revealed via text-to-speech or voice notes, allowing users to revisit cherished memories. Combining cognitive stimulation with memory recall, Ladder enriches daily experiences in an interactive, meaningful way.
